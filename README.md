## 🔍 ¿Qué es este sistema?

Plataforma web desarrollada para que los estudiantes de **1° a 5° año (secciones GHC y FSN)** de la Unidad Educativa *"Tobogán de la Selva"* consulten de forma segura y personalizada sus calificaciones en la actividad:

> 📝 **ACTIVIDAD PRÁCTICA (FORMULARIO)**  
> 🗓️ Tercer Lapso - Periodo Escolar 2024-2025

🔗 **Acceso directo**: https://edwardcarrasco.github.io/consulta-de.notas-ghc-y.fsn/

---

## ✨ Características

| Funcionalidad | Descripción |
|--------------|-------------|
| 🔐 Búsqueda por cédula | Los estudiantes ingresan su cédula para ver **únicamente su nota** |
| 📱 Diseño responsive | Funciona en celulares, tablets y computadoras |
| ⚡ Carga rápida | Datos en formato JSON para consultas instantáneas |
| 🎨 Interfaz educativa | Diseño amigable con colores institucionales y emojis motivacionales |
| 🔒 Privacidad protegida | No muestra listas completas ni datos sensibles de otros estudiantes |
| 🇻🇪 Contexto bolivariano | Lenguaje pedagógico alineado con valores revolucionarios |
